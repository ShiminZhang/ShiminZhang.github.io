pipeline of scheduling wip

Created2023-02-01|Updated2023-02-05
|Post View:

现在是2月5日22.50pm
Week1 Day8（因为节后算8天）

原计划协同agenda，hexo，todo，格志，四个记录软件去做工作生活的记录流
希望一个工作的流程是：agenda记录思考和拆解-> todo记录具体目标和节点 -> hexo记录反思，格志全程伴随记录心路和平复心情
实际上发生的：agenda记录思考和拆解->没做todo->来不及hexo反思, 格志拉满
原因：做一步看一步，没有整体规划
周末去了趟中山，用掉一天
收拾家，招待朋友过元宵，用掉一天

可以改进的地方

  1. 无法随时随地去hexo写正经记录，需要想办法处理
  2. 家里要减慢搞乱的速度和收拾的效率，主要是衣服乱扔的问题，以后每天的衣服要分成 全新，要洗，穿过但暂时不洗三种状态，分别放在衣柜，沙发，洗衣机上
    尽量少买点东西，盒子太多真的很麻烦
  3. 再简单的工作也要做记录
  4. 考虑买台mbp，随时带着旅游的时候也能掏出来。要不天天背包带着公司这台？
    可它不是我的，我好不爽，天天背来背去的也好麻烦
    买个新的好贵啊草
  5. 考虑找一找在家以外的地方也能充分休息的办法？

做的不错的地方

  1. 起码agenda慢慢上手了
  2. 中山-虎门-黄埔 1/3 done
  3. 家收拾的不错，起码能见人
  4. 对招待客人逐渐熟悉
  5. 即使发现来不及写richtext文档了还是写了这篇记录
  6. 通过格志帮我理解我的情绪越来越熟练了，内耗显著减少

这周的收获：

  1. 看了流浪地球，理解了传承的价值。
  2. 在中山意识到，做任何career都是可以有职业素养的，工程师要严谨务实求优化，将军要鼓舞士气运筹帷幄，探险家要方向明确勇于善于面对未知。
  3. 看完源码就要做好记录，不然很容易忘掉
  4. 人永远是趋利避害的，区别在于他的自我概念是否扩散到外界（认不认为别人的利益也重要），以及对于利害的认知是否高明（目光长远深刻）
